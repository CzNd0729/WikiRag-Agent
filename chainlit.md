# WikiRag-Agent 智能助手

欢迎使用 **WikiRag-Agent**！这是一个基于 **LangGraph** 和 **MCP (Model Context Protocol)** 构建的高性能百科助手。

### 核心功能
- **百科检索**：基于 RAG 技术，精准回答百科类问题。
- **长短期记忆**：支持用户画像与对话上下文持久化。
- **实时工具调用**：集成 MCP 协议，支持实时数据获取。
- **反思优化**：通过多步思考与反思，提供更高质量的答案。

### 常用操作
- 直接输入您想查询的问题。
- 系统会自动调用相关工具进行检索。
- 在左侧侧边栏可以管理您的对话记录。
